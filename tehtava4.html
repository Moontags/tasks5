JavaScript Local Storage Example

What is Local Storage in relation to JavaScript?

localStorage is part of the Web Storage API that allows you to store key-value pairs in the browser's storage. The data persists across browser sessions, meaning it doesn't get deleted when the browser is closed. localStorage is scoped to the origin (domain, protocol, and port) of the script that loaded the page.

How can you use it?

The methods of the localStorage object include:

  • setItem(key, value): Stores a key-value pair.
  • getItem(key): Retrieves the value for a given key.
  • removeItem(key): Removes a key-value pair.
  • clear(): Clears all stored key-value pairs.
  • key(index): Returns the key at a given index.
  • length: Returns the number of stored key-value pairs.
